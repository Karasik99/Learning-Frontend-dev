Требования:
Для создания контейнера карточки используйте один из семантических тегов (section или article)
Карточка должна иметь рамку шириной в 1px, быть серого цвета (используйте способ задания цвета через использование его непосредственного названия) и иметь сплошной стиль
Ширина карточки должна быть равна 300px, а высота карточки должна быть равна 460px
Карточка должна иметь внешний отступ сверху в 5px и отступ слева в 10px
Внутренние отступы карточки должны быть равны 50px
Текст внутри карточки должен быть выравнен по центру
Для добавления заголовка карточки используйте один из тегов заголовков и задайте ему размер текста в 18px, для добавления остального текста используйте тег параграфа и тег текста и укажите размер в 14px
Цвет текста с ценой и валютой можно задать любой, размер текста цены должен быть равен 48px, а размер текста валюты должен быть равен 18px
Для добавления кнопки "Заказать услугу" можете использовать универсальный тег блочного элемента либо самостоятельно изучить как работать с тегом создания кнопок button и использовать его
В случае создания кнопки через тег div:
Задайте кнопке такую же рамку, как и карточке, но шириной в 2px
Внутренние отступы кнопки должны быть равны 20px сверху и снизу и 30px справа и слева
Текст внутри кнопки должен быть весь заглавными буквами, иметь размер в 16px, иметь полужирное начертание и должен быть выравнен по центру
В случае создания кнопки через тег button:
Все стили кнопки по умолчанию должны быть сброшены
Рамка кнопки должна быть такой же, как и у карточки, но иметь ширину в 2px
Внутренние отступы кнопки должны быть равны 20px сверху и снизу и 30px справа и слева
Текст внутри кнопки должен быть весь заглавными буквами, иметь размер в 16px, иметь полужирное начертание и должен быть выравнен по центру
В макете используется нестандартный шрифт. При желании можно дополнительно его подключить. Но тут есть нюанс. На сервисе Google Fonts данного шрифта нет. Поэтому можно либо использовать вместо него шрифт MuseoModerno с сервиса, либо самостоятельно изучить способ добавления локальных шрифтов
Подсказки:
Для стилизации отдельных элементов используйте обращение по селектору класса (добавить атрибут class="element" в открывающем теге и обращаться по названию класса .element {})
Для общей стилизации одинаковых элементов используйте обращение по селектору элемента (обращение по названию тега element {})
